ETF Basket Dashboard — Quick Publish

Fastest (no code):
1) Netlify Drop
   • https://app.netlify.com/drop
   • Drag etf-site.zip into the page
   • You’ll get an instant *.netlify.app URL
   (Optional) Site Settings → Domain: set a custom subdomain.

2) Cloudflare Pages (direct upload)
   • Pages → Create a project → "Direct Upload"
   • Upload etf-site.zip
   • Get a *.pages.dev URL (custom domain optional)

3) GitHub Pages
   • New repo (e.g., advisor-dashboard)
   • Upload index.html to repo root
   • Settings → Pages → Deploy from a branch: main (root)
   • Live at https://<username>.github.io/<repo>/
